# Retail-Customer-Profiling-and-Behavioral-Segmentation

## Project Overview
This project performs **customer segmentation** on the **UCI Online Retail Dataset** using data analysis and machine learning techniques.  
The goal is to group customers based on purchasing behavior to support **targeted marketing, customer retention, and business strategy**.

---

## Dataset
- Source: UCI Machine Learning Repository
- Name: Online Retail Dataset
- Description:  
  Transaction-level data from an online retail store including:
  - InvoiceNo
  - StockCode
  - Description
  - Quantity
  - InvoiceDate
  - UnitPrice
  - CustomerID
  - Country

---

## Project Workflow
1. Data Cleaning  
   - Removed missing values  
   - Filtered canceled transactions  
   - Converted data types  

2. Exploratory Data Analysis (EDA)  
   - Customer purchase patterns  
   - Sales and frequency analysis  
   - Outlier detection  

3. Feature Engineering  
   - Customer-level aggregation  
   - Behavioral metrics creation  

4. Customer Segmentation  
   - Applied K-Means clustering  
   - Determined optimal number of clusters  
   - Assigned customers to segments  

5. Insights  
   - Identified high-value and low-engagement customers  
   - Derived business-friendly interpretations
